# Password Strength Checker

## Project Overview

The Password Strength Checker is a Python cybersecurity project that checks the strength of a user's password. The program checks whether the password meets common security requirements and classifies it as Weak, Medium, Strong, or Very Strong.


---

## Problem Statement

Many users create weak passwords that are easy to guess or crack. Weak passwords can lead to unauthorized access to personal accounts and sensitive information.

This project helps users understand password security by checking password strength and providing feedback for improvement.

---

## Features

* Checks minimum password length
* Detects uppercase letters
* Detects lowercase letters
* Detects digits
* Detects special characters
* Detects common weak passwords
* Calculates a password strength score
* Classifies passwords as Weak, Medium, Strong, or Very Strong
* Provides suggestions for improving password security
